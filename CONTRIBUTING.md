# Contributing to Indro-LLM-Trainer

We welcome contributions to Indro-LLM-Trainer! Please follow these guidelines to make the contribution process smooth and easy for everyone.

## How to Contribute

1. **Fork the Repository**  
   Start by forking the repository on GitHub. This gives you your own copy to work on.

2. **Clone Your Fork**  
   Clone your forked repository to your local machine using:
   ```bash
   git clone https://github.com/<your-username>/Indro-LLM-Trainer.git
   ```

3. **Create a Branch**  
   Create a new branch for your feature or bug fix:
   ```bash
   git checkout -b my-feature-branch
   ```

4. **Make Your Changes**  
   Make your changes in the codebase.

5. **Commit Your Changes**  
   Commit your changes using:
   ```bash
   git commit -m "Add a descriptive commit message"
   ```

6. **Push Your Changes**  
   Push your changes back to your fork:
   ```bash
   git push origin my-feature-branch
   ```

7. **Open a Pull Request**  
   Go to the original repository and open a pull request. Describe your changes and why they are valuable.

## Guidelines  
- Ensure your code follows the project's coding standards.  
- Write tests to ensure your changes work as expected.  
- Update any documentation if your changes affect it.  
- Be respectful and collaborative in discussions.

## Code of Conduct  
Please make sure to read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).  
We expect all contributors to treat each other with respect and kindness.

Thank you for considering contributing to Indro-LLM-Trainer! Your contributions are greatly valued.
